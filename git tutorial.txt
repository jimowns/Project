Bestanden toevoegen in git

git init
git pull --rebase git@github.com:jimowns/Project.git
git add New\ Text\ Document.txt
git commit -m 'dit is een commit'
git remote add origin git@github.com:jimowns/Project.git
git push origin master


Bestanden verwijderen in git

git rm file1.txt
git commit -m "remove file1.txt"
git push -u
